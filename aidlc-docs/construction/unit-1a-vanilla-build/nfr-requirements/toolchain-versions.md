## Note on `msys2-devel`
The plan's requested package `msys2-devel` is not an installed package itself — it's a virtual name `Provides`'d by the `gcc` package in this MSYS2 repository state. `pacman -S msys2-devel` correctly resolved to installing `gcc` (already listed above). Confirmed via `pacman -Si msys2-devel` (not found as a real package) and `pacman -Qi gcc | grep Provides` (shows `msys2-devel`).
